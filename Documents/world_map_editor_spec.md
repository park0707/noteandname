# 노벨플로우 세계관 지도 편집기 기능 상세 설계서
(Novelflow World Map Editor Feature Specification)

이 문서는 웹소설 작가를 위한 설정 오류 방지 및 창작 어시스턴트 플랫폼인 **노벨플로우(Novelflow)**의 새로운 핵심 기능인 **세계관 지도 편집기(World Map Editor)**의 상세 기능 및 구현 규격을 정의합니다.

---

## 1. 기능 개요 및 설계 철학

### 1.1 기능 개요
판타지, 무협, 이세계 등 가상 세계를 배경으로 하는 웹소설 작가가 지도를 직접 창작하고, 그 위에 서사 정보(인물 위치, 세력권 변화, 복선 설정 장소 등)를 연결 및 관리할 수 있는 인터랙티브 지도 편집 기능입니다. 단순한 "이미지 업로드 후 핀 배치"를 넘어, 작가가 캔버스 위에 직접 영역을 그리고 선을 연결하며 서사 데이터와 유기적으로 소통할 수 있는 지도 제작기를 목표로 합니다.

### 1.2 핵심 설계 철학
> **"지도는 단순한 그림이 아니라 서사 관리 도구다."**
- 예쁜 지도를 만드는 디자인 툴이 아닌, 글을 쓸 때 세계관의 지리적 일관성을 유지하고 인물의 동선과 세력 변화를 시각적으로 파악할 수 있도록 돕는 도구입니다.
- 모든 지도 요소는 작품의 회차(Chapters), 캐릭터(Characters), 복선(Foreshadowing), 역사적 사건(History)과 긴밀히 연결됩니다.

## 2. 지도 구조: 계층형 드릴다운 (Nested Map)

지도는 단일 평면이 아닌 트리 구조의 계층형 지도로 구성되어 공간의 깊이를 표현합니다.

```
[세계 지도] (전체 대륙, 왕국, 산맥, 바다 위치 표시)
  ├── [아이론 왕국] (더블클릭 시 진입)
  │     ├── [수도 아이론시]
  │     │     └── [왕궁 내부도]
  │     ├── [북부 산악지대]
  │     └── [동부 국경 요새]
  ├── [흑룡 산맥]
  └── [남방 제국]
```

### 2.1 계층 간 이동 UX
- **진입**: 폴리곤(영역) 또는 핀(포인트)을 더블클릭하면 해당 요소의 하위 지도로 이동합니다.
- **복귀**: 에디터 상단의 빵 부스러기(Breadcrumb) 네비게이션을 사용하여 상위 계층으로 이동합니다.
  - *예: 세계 지도 > 아이론 왕국 > 수도 아이론시*
- **무제한 계층 구조**: 계층의 수에 제한이 없으므로 대륙 → 왕국 → 도시 → 건물 → 내부 방 구조까지 무한히 계층화할 수 있습니다.

## 3. 지도 편집 도구 (Map Drawing Tools)

### 3.1 영역 그리기 (Polygon Drawing)
작가가 마우스로 직접 영역의 경계를 그리고 지정합니다.
- **폴리곤 도구**: 클릭하여 꼭짓점을 하나씩 찍어 영역을 완성합니다. 첫 번째 꼭짓점을 다시 클릭하면 영역이 닫히며 생성됩니다.
- **자유 드로잉 도구**: 마우스를 누른 채 드래그하여 자유 형태의 경계 영역을 생성합니다.
- **꼭짓점 편집**: 영역 완성 후 각 꼭짓점을 마우스 드래그로 이동하여 세부 형태를 수정할 수 있습니다.
- **영역 속성**:
  - **이름**: 영역명 (예: 아이론 왕국, 흑룡 산맥 등)
  - **유형**: 왕국 / 제국 / 산맥 / 숲 / 바다 / 사막 / 금지구역 / 기타
  - **색상**: 유형별 기본 템플릿 색상 제공 및 커스텀 컬러 선택기 지원
  - **투명도**: 0~100% 조절 (영역 겹침 및 레이어 표현용)
  - **설명**: 간략한 세계관 설명 텍스트
  - **하위 지도 연결**: 영역 더블클릭 시 열릴 하위 지도를 설정합니다.

#### 유형별 기본 색상 팔레트
| 유형 | 기본 색상 |
| :--- | :--- |
| **왕국 / 제국** | 연한 황색 계열 |
| **산맥** | 회갈색 |
| **숲 / 정글** | 초록 계열 |
| **바다 / 강** | 파란 계열 |
| **사막** | 모래색 |
| **금지구역** | 붉은 계열 |
| **세력 미정** | 회색 |

### 3.2 포인트(핀) 배치
지도상 단일 위치를 표시하는 마커 기능입니다.
- **핀 유형**:
  - 도시 / 마을 / 소도시
  - 성 / 요새 / 궁전
  - 유적 / 던전 / 비밀 장소
  - 전투 발생 지점
  - 캐릭터 현재 위치 (캐릭터 추적용으로 별도 관리 가능)
  - 커스텀 (아이콘 직접 선택)
- **핀 속성**:
  - 이름, 유형, 아이콘 선택
  - 설명 텍스트 (마크다운 포맷 지원)
  - 하위 지도 연결 (클릭 시 이동할 타깃 맵)
  - 관련 회차 연결 (등장 에피소드)
  - 관련 캐릭터 연결 (인물 매핑)

### 3.3 선 (Route / Border) 그리기
- **경로 선**: 캐릭터의 이동 경로, 주요 교역로, 군사 순찰로 등을 선으로 표시합니다.
- **국경선**: 영역과 별개로 세력 간의 경계선을 선으로 강조하여 표시합니다.
- **스타일 옵션**: 선 색상, 두께, 스타일(실선/점선/파선) 설정 가능.
- **레이블 표시**: 양 끝단 및 경로 중앙에 텍스트 레이블 표시 가능 (예: *"왕도 → 국경 요새 / 3일 거리"*).

### 3.4 텍스트 레이블
- 캔버스 위 자유 위치에 텍스트를 직접 추가할 수 있습니다.
- 폰트 크기, 색상, 회전 각도 조절이 가능하여 지명 표기 대형 텍스트 및 지도 주석 등에 활용됩니다.

## 4. 지도 히스토리 및 타임라인 (Map Timeline)

소설의 전개 및 시간의 흐름(회차, 사건 등)에 따라 지도의 형태와 상태가 동적으로 변화하는 것을 관리하는 핵심 기능입니다.

### 4.1 히스토리 스냅샷
- 특정 시점(특정 회차, 주요 사건 기준)의 지도 상태를 스냅샷으로 저장합니다.
  - *예: "1권 시작 기준", "아이론 왕국 멸망 직후", "최종화 기준"*
- 각 스냅샷마다 작중 날짜(캘린더 연동 가능) 및 요약 메모를 기록할 수 있습니다.

### 4.2 타임라인 슬라이더
- 지도 화면 하단에 가로형 타임라인 슬라이더를 제공합니다.
- 슬라이더를 드래그하여 과거/미래 시점으로 바를 이동하면, 해당 시점의 지도 스냅샷 상태로 캔버스가 동적 전환됩니다.
- 이전 시점 대비 변경된 영역(세력 병합 등)은 화면에서 깜빡임(Highlight) 효과로 시각적으로 강조됩니다.

### 4.3 변경 이력 추적 (Change Log)
- 각 지도 요소(영역, 핀, 선)마다 변경 이력이 역사 기록처럼 누적 저장됩니다.
- **예시**:
  - *1권 3화*: 아이론 왕국 영토 확장 (동부 국경선 이동)
  - *2권 1화*: 흑룡 산맥 내 비밀 유적 핀 추가
  - *2권 7화*: 북부 영토 제국에 병합 (영역 색상 연한 황색 -> 붉은색 변경)

## 5. 캐릭터 위치 추적 (Character Tracking)

캐릭터를 지도 위에 배치하여 실시간 동선과 현재 위치를 입체적으로 모니터링합니다.

### 5.1 캐릭터 마커
- 캐릭터 설정(기존 캐릭터 프로필 기능)과 연동되어 작동합니다.
- 캐릭터별 고유 색상 및 미니 프로필 사진이 들어간 마커가 자동 생성됩니다.
- 마커 클릭 시: 캐릭터 이름, 현재 상태(예: 부상, 수련 중), 소속 세력, 최근 등장 회차를 간편히 확인 가능합니다.

### 5.2 동선 추적
- 회차가 진행됨에 따라 작가가 캐릭터의 위치를 지도 위에 실시간으로 업데이트합니다.
- 지난 이동 경로를 점선(Dotted line)으로 시각화할 수 있습니다 (옵션으로 켜고 끄기 지원).
- **"주인공은 현재 어디에 있나?"**를 지도상에서 한눈에 찾을 수 있는 단축 조회를 제공합니다.

### 5.3 다중 캐릭터 동시 표시 및 필터링
- 특정 회차 시점을 기준으로 소설에 등장하는 모든 캐릭터의 위치를 한 지도 위에서 조율합니다.
- 필터링 기능: 주인공 그룹만 표시, 적대 세력만 표시, 특정 원정 파티원만 표시 등의 커스텀 필터링을 지원합니다.

## 6. 서사 연결 기능 (Narrative Integration)

지도 위의 시각적 요소들을 소설 본문 및 플롯 데이터와 융합합니다.

### 6.1 복선(Foreshadowing) 연동
- 기존 복선 관리 기능과 통합하여 작동합니다.
- 지도상의 핀(장소)에 복선을 연동하면, 해당 핀에서 복선 정보 카드(예: *"이 유적은 12화 복선과 연결됨"*)로 즉시 접근이 가능합니다.
- 복선의 상태(미회수/회수됨)를 지도 위 아이콘 스타일의 차이로 직관적으로 표시합니다.

### 6.2 회차(Chapters) 연동 및 양방향 이동
- 특정 장소의 핀을 클릭하면, 해당 장소가 등장하거나 언급된 **회차 목록**이 팝업으로 자동 정렬됩니다.
- 반대로, 소설 에디터 본문 내에서 특정 장소 이름이 언급되거나 태그되었을 때 클릭하면, 해당 지도의 핀 위치로 화면이 자동 포커싱되며 드릴다운(Zoom-in) 이동합니다.

## 7. 저장 및 내보내기 사양 (Data Schema & Export)

### 7.1 GeoJSON 기반 커스텀 JSON 스키마
지도 데이터는 공간 좌표 정보와 소설 속성을 결합한 확장형 GeoJSON 포맷으로 저장됩니다.

```json
{
  "mapId": "world-map-001",
  "title": "아이론 대륙",
  "layers": [
    {
      "id": "region-001",
      "type": "polygon",
      "name": "아이론 왕국",
      "category": "kingdom",
      "color": "#f5e6a3",
      "coordinates": [[100, 200], [300, 200], [350, 400], [150, 420]],
      "childMapId": "aeron-kingdom-map",
      "linkedChapters": [1, 3, 7],
      "history": [
        { "snapshotId": "vol1-start", "visible": true, "color": "#f5e6a3" },
        { "snapshotId": "vol2-after-war", "visible": false }
      ]
    }
  ],
  "characters": [
    {
      "characterId": "char-hero-001",
      "currentPosition": [220, 310],
      "trail": [[100, 100], [180, 240], [220, 310]]
    }
  ]
}
```

### 7.2 내보내기 옵션
- **PNG (고해상도)**: 작가의 개인 소장 및 연재처 공지, 독자 공개용 고품질 이미지 변환.
- **JSON**: 지도 데이터 전체 백업 및 다른 계정/기기로의 이전용.
- **공유 링크 (읽기 전용)**: 편집 권한 없이 지도와 캐릭터 동선을 타인(편집자, 베타리더, 독자)에게 웹 뷰어로 링크 공유.

## 8. 기술 구현 스택 및 아키텍처

현재 노벨플로우 프로젝트가 **React + TypeScript + Vite** 기반으로 구성되어 있으므로, 다음 스택을 결합하여 구현합니다.

### 8.1 프론트엔드 라이브러리 구성
- **지도 캔버스 & 좌표계**: `Leaflet.js` + `CRS.Simple`
  - 실제 지구 좌표(위도/경도)가 필요 없는 판타지 가상 좌표계 시스템에 최적이며 오픈소스 가벼움 제공.
- **폴리곤 그리기 및 꼭짓점 편집**: `Leaflet-Geoman`
  - 마우스를 이용한 자유 드로잉, 폴리곤 생성, 꼭짓점 수정 인터랙션을 기본 제공하여 개발 공수 단축.
- **React 래퍼**: `react-leaflet`
  - Leaflet을 선언적 React 컴포넌트 형태로 안전하게 렌더링.
- **이미지 캡처 및 내보내기**: `leaflet-image` + `html2canvas`
  - Leaflet 캔버스 내 레이어 전체를 하나의 PNG 이미지로 병합 변환.
- **상태 관리**: `Zustand`
  - 계층 지도 트리 정보 및 현재 뷰 계층 레벨, 타임라인 스냅샷 인덱스 상태를 중앙 집중식 관리.

### 8.2 핵심 컴포넌트 구조
React 프로젝트 내 `src/components/WorldMap/` 경로 아래에 모듈별로 격리 구현합니다.

- **`WorldMapEditor.tsx`**: 전체 에디터 화면을 조율하는 컨테이너 컴포넌트.
- **`MapCanvas.tsx`**: Leaflet 인스턴스를 생성하고 캔버스를 화면에 뿌리는 코어 래퍼.
- **`DrawingToolbar.tsx`**: 폴리곤, 핀, 경로선, 텍스트 도구를 선택하는 상단/측면 플로팅 툴바.
- **`LayerPanel.tsx`**: 배치된 모든 요소의 목록을 보여주고 가시성(Show/Hide)을 제어하는 레이어 목록 패널.
- **`MapBreadcrumb.tsx`**: 현재 지도 계층 깊이를 표현하고 상위 지도로 복귀하게 돕는 네비게이션 바.
- **`ElementPropertiesPanel.tsx`**: 선택된 지도 요소(영역/핀/선)의 이름, 설명, 연결 설정(캐릭터, 회차)을 수정하는 속성 편집 창.
- **`CharacterOverlay.tsx`**: 캐릭터의 위치를 마커 형태로 매핑하여 Canvas 위에 그리는 캐릭터 오버레이 레이어.
- **`HistoryTimeline.tsx`**: 화면 최하단에서 타임라인 스냅샷 간 이동을 관할하는 슬라이더 컴포넌트.
- **`MapExportMenu.tsx`**: PNG 이미지 스크린샷 렌더링 및 JSON 데이터 추출 메뉴.
