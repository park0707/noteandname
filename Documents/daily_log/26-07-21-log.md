## 3. 🐛 이슈 트래킹 및 문제 해결 (Troubleshooting & Resolution)

### [이슈 1] 격자 스냅 켜짐 상태에서 드래그 시 좌표가 지나치게 튀는 현상
- **증상 (Symptom)**: 요소를 드래그할 때 줌 배율(zoomLevel)에 따라 그리드 스냅 연산 단위가 어긋나며 위치가 갑자기 멀리 튐
- **원인 분석 (Root Cause)**: 마우스 dx, dy 변위에 스냅 연산을 적용할 때 줌 배율 역수 처리와 스냅 반올림(Math.round) 순서가 어긋남
- **해결 방식 (Solution)**: 마우스 델타값을 뷰포트 좌표계로 변환한 후 absolute 좌표에 스냅 연산을 수행하도록 보정함
- **수정 위치**: [useGridSnap.ts](file:///c:/project/noteandname/src/hooks/useGridSnap.ts)
- **핵심 코드 스니펫**:
```ts
// ✅ 올바른 그리드 스냅 연산
export const snapToGrid = (value: number, gridSize: number, enabled: boolean): number => {
  if (!enabled || gridSize <= 0) return value;
  return Math.round(value / gridSize) * gridSize;
};
```
