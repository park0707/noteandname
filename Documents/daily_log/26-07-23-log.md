## 3. 🐛 이슈 트래킹 및 문제 해결 (Troubleshooting & Resolution)

### [이슈 1] 붓(Brush) 그리기 영역 이동 시 테두리만 움직이고 붓 본체가 제자리에 남는 문제
- **증상 (Symptom)**: 붓 요소를 선택하여 드래그 이동하면 바운딩 박스 테두리만 마우스를 따라가고 붓으로 그린 실제 선(Stroke)은 캔버스에 멈춰있음
- **원인 분석 (Root Cause)**: `handleCanvasMouseMove`의 드래그 연산에서 `brushStrokes` 배열에만 변위가 적용되고 내부 개별 stroke 객체들(`brushStrokeObjects`) 백업 및 offset 갱신이 누락되었음
- **해결 방식 (Solution)**: `initialCoords` 백업에 `brushStrokeObjects`를 포함시키고 마우스 이동 변위(`finalDx, finalDy`)를 stroke 지점 좌표 전체에 더하도록 수정함
- **수정 위치**: [WorldMapCanvas.tsx](file:///c:/project/noteandname/src/components/worldview/WorldMapCanvas.tsx#L450-L480)
- **핵심 코드 스니펫**:
```tsx
// ✅ 붓 드로잉 궤적 좌표 변위 연산 보정
if (element.type === 'brush' && initial.brushStrokeObjects) {
  updatedElement.brushStrokeObjects = initial.brushStrokeObjects.map(obj => ({
    ...obj,
    points: obj.points.map(p => ({
      x: p.x + finalDx,
      y: p.y + finalDy
    }))
  }));
}
```
